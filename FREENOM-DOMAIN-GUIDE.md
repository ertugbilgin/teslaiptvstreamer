# 🆓 Freenom'dan Ücretsiz Domain Alma

`.tk`, `.ml`, `.ga`, `.cf`, `.gq` uzantıları 12 ay ücretsiz!

---

## 📝 Adım 1: Hesap Oluşturma

1. **https://freenom.com** adresine gidin
2. Sağ üstten **"Sign Up"** tıklayın
3. E-posta adresinizi girin
4. Doğrulama linkine tıklayın
5. Şifre belirleyin

---

## 🔍 Adım 2: Domain Ara

1. Ana sayfadaki arama kutusuna yazın:
   ```
   tesla-iptv
   ```
   veya
   ```
   tv-stream
   ```
   veya
   ```
   my-iptv
   ```

2. **"Check Availability"** tıklayın

3. Yeşil tik ✓ olanlar müsait!

---

## ✅ Adım 3: Domain Seç

Müsait uzantılar görünecek:
```
✓ TESLA-IPTV.TK   - FREE
✓ TESLA-IPTV.ML   - FREE
✓ TESLA-IPTV.GA   - FREE
✗ TESLA-IPTV.COM  - NOT AVAILABLE
```

**Tavsiye:** `.tk` veya `.ml` seçin (daha güvenilir)

**"Get it now"** → **"Checkout"**

---

## 🛒 Adım 4: Sipariş Tamamla

1. **Period:** 12 Months @ FREE seçin
2. **"Continue"** tıklayın
3. E-posta doğrulaması yapın (gelmiş olmalı)
4. **"Complete Order"**

---

## 🌐 Adım 5: Nameserver Ayarı (Cloudflare için)

Domaininiz aktif olduktan sonra:

1. Freenom Dashboard → **Services** → **My Domains**
2. Domaininizin yanındaki **"Manage Domain"** tıklayın
3. **Management Tools** → **Nameservers**
4. **"Use custom nameservers"** seçin
5. Cloudflare verdiği nameserver'ları yazın:
   ```
   lara.ns.cloudflare.com
   greg.ns.cloudflare.com
   ```
   (Cloudflare size özel verecek, bunlar örnek)

6. **"Change Nameservers"**

⚠️ **Bu değişiklik 24 saate kadar sürebilir!**
